# Introduction

**omriss** (Omriss Editor) helps you clarify ideas and refine them,
consideration by consideration — *layer by layer*.

A Markdown document is more than a wall of text: its headings form a stack of
layers. omriss makes that structure the way you work. The outline is your
map; each section is a layer you can focus on alone, refine, and leave —
confident that nothing else in the file changed.
