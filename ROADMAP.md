# Roadmap

The authoritative roadmap, milestone definitions (M0–M9), and theme
breakdown live in the design documents:

- RFC index: [`rfcs/README.md`](rfcs/README.md)
- Lifecycle policy: [`rfcs/done/000-rfc-lifecycle-policy.md`](rfcs/done/000-rfc-lifecycle-policy.md)

Status snapshot (v0.13.0): all 47 RFCs (000–046) implemented; M0–M9 themes
complete. The desktop MVP runs on Linux/macOS/Windows (the desktop shell
requires platform WebView libraries to build — see `PLATFORMS.md`). The
outline uses the `dioxus-swdir-tree` widget. Five future concepts
(multi-document, plugins, AI, visual graph, TUI/CLI) remain roadmap ideas
only, with no formal RFCs written. `rfcs/proposed/` is currently empty.
